## üõ† Technical Blueprint

The audio engine is built entirely with the **Web Audio API**, using mathematical synthesis rather than static samples to ensure every note feels alive.

* **Additive Synthesis:** Layers three distinct sine waves to recreate the unique "hollow" harmonic resonance of a real bamboo flute.
* **Portamento Glide:** Implements a 100ms frequency transition (linear ramp) to simulate the fluid movement of air within a physical pipe.
* **Custom Reverb:** Uses a convolution engine with a generated impulse response to place the sound in a virtual 2.5-second hall.
* **Dynamic Visuals:** A real-time aura visualizer built on the **Canvas API**, mapped to the frequency data of the master output.

## üéÆ How to Play

1.  **Initialize:** Click the "Initialize Engine" button to start the AudioContext.
2.  **Breathe:** Hold the **SPACEBAR** to blow air into the instrument.
3.  **Notes:** Use keys **Q, W, E, R, T, Y** to cover and uncover the flute holes.
4.  **Flow:** Experiment with releasing keys while holding space to hear the portamento glide.
